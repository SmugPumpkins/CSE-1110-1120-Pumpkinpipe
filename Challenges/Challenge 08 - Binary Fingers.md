# Challenge 08 - Binary Fingers

The purpose of this challenge is to explore computer science topics such as binary numbers, as well as comparing the x and y positions of different hands being detected.

## Mild 🌶️

Create a program that meets the following criteria:

* A live video feed is captured throug a webcam.
* Hands are detected.
* Hands are drawn.
* Use the finger flags of 2 hands to get a binary number of fingers being up and down.
* Display the binary number in the top left corner of the screen.
* Display the binary number converted to base 10 beneath the binary number (this may require some additional research, but python hs built in ways of doing this or you can create a custom algorithm).

## Medium 🌶️🌶️

Create a program that meets the following criteria:

* A live video feed is captured throug a webcam.
* Hands are detected.
* Hands are drawn.
* Use the finger flags of 2 hands to get a binary number of fingers being up and down.
* Compare the x positions of the hands to make sure the hand to the left of the screen has its binary number added to the binary string before the hand on the right most side of the screen.
* Compare landmarks on the hand so that it orders the binary digits to match the order appearing on the hand (for example, if the pinky is on the left side of the hand on the screen, then the binary order should be pinky, ring, middle, index, thumb).
* Display the binary number in the top left corner of the screen.
* Display the binary number converted to base 10 beneath the binary number (this may require some additional research, but python hs built in ways of doing this or you can create a custom algorithm).
