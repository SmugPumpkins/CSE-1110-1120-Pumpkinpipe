# Challenge 03 - Tap Detection

The purpose of this challenge is to practice dealing with event loop timin as well as landmark distances on the hands.

## Mild 🌶️

Create a program that meets the following criteria:

* A live video feed is captured throug a webcam.
* Hands are detected.
* Hands are drawn.
* When the index finger tip and the thumb tip are put together, the word "tap" is displayed as text on the screen.

## Medium 🌶️🌶️

Create a program that meets the following criteria:

* A live video feed is captured throug a webcam.
* Hands are detected.
* Hands are drawn.
* When any finger tip and the thumb tip are put together, the word "tap" is displayed as text on the screen with the name of the finger being tapped (for example, `index`, `middle`, `ring`, or `pinky`).

## Spicy 🌶️🌶️🌶️

Create a program that meets the following criteria:

* A live video feed is captured throug a webcam.
* Hands are detected.
* Hands are drawn.
* A counter starting at 0 is displayed in the top left corner of the screen.
* Whenever the thumb and the index finger are first touched together, the counter increases by 1.
* The counter does not increase while the fingers are held together, only on a new "tap".
