# Lesson 01 - Setup and Installation

## Installation

### Option 1: `pip install pumpkinpipe`

1. Open Terminal, Command Prompt, or PowerShell.
2. Type `pip install pumpkinpipe`.
3. Press Enter.
4. Wait for the installation to finish before continuing.

### Option 2: Using the Python Package Manager in PyCharm

1. Open your project in PyCharm.
2. Open **Settings**.
3. Select **Project: *your project name***.
4. Select **Python Interpreter**.
5. Click the **+** button.
6. Search for `pumpkinpipe`.
7. Select `pumpkinpipe` from the results.
8. Click **Install Package**.
9. Wait for the installation to finish before continuing.
