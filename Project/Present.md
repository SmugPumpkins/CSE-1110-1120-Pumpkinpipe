# Pumpkinpipe - Present

## Overview

Share something phenomenal using the Pumpkinpipe library and Python. Your project will need to be planned, tested, and presented to the class to demonstrate your understanding of structured programming. Be sure to focus on using `selection` and `iteration` structures to make your project work.

## Project Criteria

Your task is to:

* Share the project you built with Pumpkinpipe and Python.
* Demonstrate how users can interact with the project.
* Explain your program:
    * Explain the IPO modules your project includes:
        * Explain the inputs you chose to use.
        * Explain how data from the inputs are processed.
        * Explain how the program outputs to the user.
    * Explain how you used selection control structures.
    * Explain how you used iteration control structures.

### Demonstration Video

Record a video no longer than 1 minute in length demonstrating what your project does. The video does not need to be a studio level production, but it should clearly show what inputs are used to control your project, and the outputs produced. For windows, screen recording can be accessed with the command `Windows` + `Shift` + `S` and selecting the video mode.
